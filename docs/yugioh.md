# Yu-Gi-Oh! — Collectors Plugin

> 🚧 **Not yet implemented.** This page documents the planned support.

---

## Status

Yu-Gi-Oh! support is planned for a future version. The game tab is visible in the plugin settings and the New Collection modal (when enabled), but collection creation is not functional yet.

---

## Planned Features

- Set collections by Yu-Gi-Oh! set code
- Variant support (1st edition / unlimited / limited)
- Price integration (source TBD)
- Card images and data source TBD

---

## Enabling / Disabling the Tab

The Yu-Gi-Oh! tab in the New Collection modal can be toggled in:

**Settings → Collectors → Yu-Gi-Oh!** → Enable Yu-Gi-Oh!

Disabling it hides the tab from the collection wizard — it has no effect on any existing files.
