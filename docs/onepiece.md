# One Piece TCG — Collectors Plugin

> 🚧 **Not yet implemented.** This page documents the planned support.

---

## Status

One Piece TCG support is planned for a future version. The game tab is visible in the plugin settings and the New Collection modal (when enabled), but collection creation is not functional yet.

---

## Planned Features

- Set collections by One Piece TCG set code
- Variant support (normal / parallel / secret rare equivalents)
- Price integration (source TBD)
- Card images and data source TBD

---

## Enabling / Disabling the Tab

The One Piece tab in the New Collection modal can be toggled in:

**Settings → Collectors → One Piece** → Enable One Piece

Disabling it hides the tab from the collection wizard — it has no effect on any existing files.
